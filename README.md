# Rust-Disaster-Recovery

![CI](https://github.com/skylerblue333/Rust-Disaster-Recovery/workflows/CI/badge.svg)

High-performance, memory-safe Rust service for recovery operations.

## Features
- Zero-cost abstractions using Actix-Web
- Thread-safe Mutex state management
- Dockerized minimal footprint deployment

## Quick Start
```bash
cargo test
cargo run
curl http://localhost:8080/health
```
